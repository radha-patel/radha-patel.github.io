Portable Password Manager

6.115 Microcomputer Project Laboratory Final Project

Secure passwords are difficult to remember and entering them is tedious. Web-based or desktop password managers are perhaps helpful, but they require a login or download on each device that a user utilizes and also have potential to be hacked. My solution - a portable, hand-held password manager, creating using a PSoC 5LP.

This device enables a user to store their usernames and passwords, and autofills on any computer when connected via USB. I programmed the PSoC 5LP to emulate the USB HID mode of a keyboard and constructed a package to convert characters to key-press input reports to enable the autofill functionality.
